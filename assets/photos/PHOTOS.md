# Photos — drop-in guide

Save the café's photos into this folder (`assets/photos/`) using the **exact
filenames** below. They appear on the site automatically — no code changes needed.
Until a file is added, the site shows a tidy "Add filename.jpg" placeholder in that slot.

- Format: **.jpg** (or .webp — if you use .webp, keep the same name, e.g. `hero.webp`,
  and tell me so I update the one reference).
- Keep files reasonably sized (long edge ~1600px, under ~400KB each) so the site stays fast.
- Landscape shots crop from the centre; portraits work best where noted.
